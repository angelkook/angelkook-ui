# angelwork-ui 组件库

### 快速开始

#### 1.安装组件库
```bash
npm i angelwork-ui
```

#### 2.引用组件库
```javascript
//全部引入
import 'angelwork-ui/dist/css/index.css';
import MUI from 'angelwork-ui';
Vue.use(MUI);

//按需引用
import 'angelwork-ui/dist/css/demo.css';
import { Demo } from "angelwork-ui";
Vue.use(Demo);
```
